# PDF-Last-Page-Print
PDF Last Page Extractor
A lightweight Python application to extract the last page from every PDF in a folder.

## Features
- Extract the last page from every PDF.
- Save each last page as a separate PDF.

Step 1- Open any code editor 
First install the library:
**pip install pypdf**

Step 2- Change the location 

# Folder containing the original PDFs
input_folder = r"C:\Users\USER\Downloads\Zone"  # Change this

# Folder where last-page PDFs will be saved
output_folder = r"C:\Users\USER\Downloads\Zone\lastpage"  # Change this

Step 3- Run the program
python app.py
